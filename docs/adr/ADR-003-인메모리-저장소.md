# ADR-003: 인메모리 저장소를 기본으로 사용

## 상태
승인 (Accepted)

## 컨텍스트
데모/교육용 프로젝트에서 데이터 저장 방식을 결정해야 한다.
- 데이터베이스 설정 없이 즉시 실행 가능해야 함
- 그러나 실제 DB 연동 데모도 필요

## 결정
Spring Profile을 활용한 이중 저장소 전략을 채택한다:
- **default 프로필**: ConcurrentHashMap 기반 인메모리 저장소
- **jpa 프로필**: Spring Data JPA + PostgreSQL 저장소
- SessionStoreInterface로 추상화하여 프로필에 따라 전환

## 결과
### 긍정적
- 외부 의존성 없이 즉시 실행 가능
- Profile 전환으로 DB 연동 데모 가능
- 테스트 격리 용이

### 부정적
- 애플리케이션 재시작 시 데이터 소실 (default)
- 두 저장소 구현 유지보수 필요
